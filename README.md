# 奶龙联络处

一个无需构建步骤的单页随机文案生成器，献给珍视奶龙的人。

👉 **在线体验:https://caigua838.github.io/ninedragon/**

## 功能

- 随机抽取奶龙主题文案
- 显示本次页面访问中的抽取次数
- 一键复制当前文案
- 适配桌面和移动端浏览
